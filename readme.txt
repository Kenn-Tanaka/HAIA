HAIA (Hybrid AI Assistant)
=========================

概要
----
HAIA はデスクトップで動作する AI アシスタントアプリケーションです。
Gemini / OpenAI / Claude などの API を利用し、
テキスト生成・分析・音声生成を行います。

本 README は、
- Python ソースコードから実行する場合
- Nuitka によりビルドされた EXE を使用する場合

の両方に対応しています。

操作方法や画面説明の詳細については、
同梱の「HAIA ユーザーズマニュアル（PDF）」を参照してください。

--------------------------------------------------
位置づけと重要な注意事項
--------------------------------------------------

HAIA は、大規模言語モデル（LLM）を利用した
**支援ツール（アシスタント）**です。

本ソフトウェアが生成するテキスト・音声・分析結果は、
最終的な判断、意思決定、公式見解を代替するものではありません。

業務利用や対外利用においては、
必ず人間による確認・検証を行ってください。

--------------------------------------------------
【1】Python 版（ソースコードから実行）
--------------------------------------------------

動作環境：
    Python 3.10 以上

任意（機能に応じて必要）：
    FFmpeg
        - MP3 生成に必要
    Pandoc
        - Word / PowerPoint 変換に必要

セットアップ：
    python -m venv .venv
    .venv\Scripts\pip install -r requirements.txt

起動方法：
    launch.bat を実行

--------------------------------------------------
【2】EXE 版（Nuitka --standalone ビルド）
--------------------------------------------------

概要：
    本バージョンは Python プログラムを Nuitka により
    --standalone オプション付きでコンパイルしたものです。

    Python ランタイムは同梱されており、
    別途 Python をインストールする必要はありません。

ディレクトリ構成：
    HAIA/
    ├─ bin/haia.exe
    ├─ *.json        （設定・データ）
    ├─ prompts/      （システムプロンプト）
    ├─ ffmpeg.exe    （任意）
    ├─ pandoc.exe    （任意）
    └─ launch_haia.cmd

※ 外部ツールは bin 直下ではなく、
  インストール先ディレクトリに配置してください。

--------------------------------------------------
生成結果に関する注意
--------------------------------------------------

LLM は高度な文章生成能力を持つ一方で、
以下の特性があります。

- 事実と異なる内容を生成する可能性
- 入力に含まれない情報の補完・推測

特に以下の用途では、必ず人間による検証を行ってください。

- 研究・学術・技術文書
- 事故・不祥事・インシデント分析
- セキュリティ・法務・医療・安全に関わる判断
- 公開資料・対外説明
