HAIA (Hybrid AI Assistant)

概要：
    デスクトップで動作するAIアシスタントアプリです。Gemini, OpenAI, ClaudeなどのAPIを利用してテキスト生成や音声対話を行います。

必要なもの：
    Python 3.10+

任意：
    FFmpeg (MP3生成に必要。haia.pyと同じフォルダに置いてください)
    Pandoc (ドキュメント変換に必要。haia.pyと同じフォルダに置いてください)

実行方法：
    以下のコマンドで仮想環境を作成し、ライブラリをインストールしてください。

    python -m venv .venv
    .venv\Scripts\pip install -r requirements.txt

    launch.bat をダブルクリックして起動します。
