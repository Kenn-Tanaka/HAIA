HAIA インストール完了

この度は HAIA をインストールいただきありがとうございます。
本ソフトウェアの全機能を利用するためには、以下の無料ツール（FFmpeg および Pandoc）の導入を推奨します。

1. 音声機能を使う場合 (FFmpeg)

MP3形式での保存や、高度な音声処理に必要です。

手順:

  1. 以下のリンクから配布サイトを開きます。
    FFmpeg ダウンロードページへ
  2. ページ内の "release builds" にある ffmpeg-release-essentials.zip をダウンロードします。
  3. ZIPを解凍し、bin フォルダの中にある ffmpeg.exe をコピーします。
  4. HAIAがインストールされたフォルダ（デスクトップのショートカットを右クリック→「ファイルの場所を開く」）に貼り付けてください。

2. Word/EPUB読込を使う場合 (Pandoc)

Wordファイルや電子書籍データの読み込みに必要です。

手順:

  1. 以下のリンクからインストーラーをダウンロードします。
    Pandoc ダウンロードページへ
  2. ページ下部の pandoc-windows-x86_64.msi をダウンロードして実行してください。
  3. 自動的に設定が完了します。
